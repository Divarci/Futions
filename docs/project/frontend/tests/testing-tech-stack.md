# Testing — Tech Stack

| Technology | Purpose |
|---|---|
| **Vitest** | Test runner and mocking (`describe`, `it`, `expect`, `vi`) |
| **@testing-library/react** | `render`, `renderHook`, `screen`, `waitFor`, `act` |
| **@testing-library/user-event** | User interaction simulation (`userEvent.click`, `userEvent.type`) |
| **@testing-library/jest-dom** | DOM assertion matchers (`toBeInTheDocument`, `toBeVisible`) |
